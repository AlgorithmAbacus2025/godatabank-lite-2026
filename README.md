# GoDataBank

**Official data. Verified evidence. Searchable insights.**

GoDataBank is a searchable repository of machine-curated evidence from official institutions.

## Mission

Collect. Verify. Organise. Summarise. Publish.

## Source Policy

GoDataBank admits only sources that satisfy all of the following conditions:

1. Official institution.
2. Public API or approved machine-readable feed.
3. English content or English metadata.
4. Stable metadata.
5. Citation metadata available.
6. Machine-readable format.
7. Public redistribution or citation permitted.

## Permanent Exclusions

GoDataBank excludes news organisations, commercial data vendors, think tanks, blogs, social media, opinion sources, and user-generated content.

## System Model

```text
Official institutions
        ↓
ALIS fetch layer
        ↓
Source archive and metadata store
        ↓
Abacus analysis layer
        ↓
Citation validator
        ↓
GoDataBank publisher
        ↓
Searchable reports and subscriber briefings
```
