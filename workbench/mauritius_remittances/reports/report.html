Findings

The World Bank series contains 31 valid annual values for Mauritius, covering 1994 to 2024. The latest valid value is for 2024, when personal remittances received were recorded as approximately 1.92% of GDP. The highest value in the valid series was recorded in 1998, at approximately 4.26% of GDP. The lowest value in the valid series was recorded in 2011, at approximately 0.004% of GDP.

Methods and Limitations

This report uses one World Bank indicator only. The data does not explain why values rose or fell. The data does not separate remittance sources by country of origin. The data does not show household-level dependence on remittances. The 2025 API record appears but has a null value, so 2024 is treated as the latest valid year.
