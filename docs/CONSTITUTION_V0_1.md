# GoDataBank Constitution v0.1

**Status:** Frozen candidate for Build Milestone 1  
**Project:** GoDataBank  
**Tagline:** Official data. Verified evidence. Searchable insights.

## 1. Identity

GoDataBank is a searchable repository of machine-curated evidence derived directly from official institutions.

GoDataBank is not a news site, opinion platform, think tank, commercial data vendor, blog, or editorial magazine.

## 2. Mission

Collect. Verify. Organise. Summarise. Publish.

## 3. Core Principle

Truth is retrieved, not generated.

All GoDataBank outputs must originate from retrieved, traceable, and verifiable source material.

## 4. Source Admission Rule

A source must satisfy all of the following conditions:

1. Official institution.
2. Public API or approved machine-readable feed.
3. English content or English metadata.
4. Stable metadata.
5. Citation metadata available.
6. Machine-readable format.
7. Public redistribution or citation permitted.

Failure of any condition results in rejection from the automated GoDataBank pipeline.

## 5. Permanent Exclusion Rule

GoDataBank permanently excludes:

- News organisations.
- Commercial data vendors.
- Think tanks.
- Blogs.
- Social media.
- Opinion platforms.
- User-generated content.

## 6. AI Agent Laws

All GoDataBank agents inherit the following directives:

- No speculation.
- No opinion.
- No unsupported prediction.
- No unsourced claims.
- No publication without validation.
- No source match means no output.

## 7. Publication Rule

Every published report must include:

- Headline.
- Summary.
- Key findings.
- Source information.
- Citations.
- Original source link.

## 8. Subscriber Principle

Subscribers receive evidence, not opinions; briefings, not articles; sources, not narratives.

## 9. Governance Rule

Any major addition must follow:

Plan → Build → Review → Reflect → Refine → Freeze → Next Milestone.
