# Citation Policy

## Purpose

The citation policy ensures that every GoDataBank output is traceable to official, admitted source material retrieved through a public API or approved machine-readable feed.

## Core Rule

No published output is allowed without citation metadata. This applies to:

- Facts
- Figures
- Tables
- Charts
- Rankings
- Comparisons
- Summaries
- Method notes
- Derived calculations

Truth is retrieved, not generated. Citation cannot be replaced by speculation, opinion, or uncited inference.

## Required Citation Metadata

Each cited source must include:

- Issuing institution
- Institution type
- Source title or dataset name
- Official source URL
- Public API, approved feed, or approved structured-file URL where applicable
- Publication date, release date, or version date where available
- Retrieval timestamp in UTC
- Version, edition, endpoint, table, field, query, or row reference where applicable
- Geographic scope where applicable
- Time period covered where applicable
- Units of measurement where applicable
- Language
- Licence or terms URL where available
- Verification status

## Claim-Level Traceability

Every report claim must be traceable to one or more admitted sources. Citation may appear at paragraph, table, chart, figure, or record level, but it must be specific enough for a reviewer to locate the evidence.

## Derived Values

Derived values must cite all input sources and state the method used. A derived value must include:

- Input values
- Input units
- Formula or transformation
- Output unit
- Time period
- Source references for each input

## Missing or Ambiguous Metadata

If required citation metadata is missing, the source may not support publication. If a source contains ambiguity, the report must state the limitation and avoid unsupported interpretation.

## Retrieval Timestamps

Retrieval timestamps must use UTC. If a report depends on a time-sensitive source, the retrieval timestamp must be close enough to the publication time to support reproducibility.

## Prohibited Citation Practices

Do not cite:

- News organisations
- Commercial data vendors
- Think tanks
- Social media
- Unofficial aggregators
- AI-generated answers
- Unsourced secondary summaries
- Blogs
- User-generated content
- Any source without public API or approved machine-readable feed access

## Publication Check

Before publication, confirm that:

- All sources are admitted.
- All citations include required metadata.
- All claims are linked to citations.
- All units and methods are stated where needed.
- No excluded source category appears in the citation list.
- No claim relies on speculation, opinion, or an unmatched source.
